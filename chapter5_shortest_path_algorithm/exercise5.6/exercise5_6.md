  (3) ループの2回目では、W = {a, d, e, f} の要素に対してbからの距離が更新される。(a <- 3, d <- 3, e <- 5, f <- 4)

  (5) ループの5回目では、W = {e, f} の要素に対してbからの距離が更新される。(e <- 4, f <- 4)

  (6) ループの6回目では、W = {f} の要素に対してbからの距離が更新される。(f <- 4)
