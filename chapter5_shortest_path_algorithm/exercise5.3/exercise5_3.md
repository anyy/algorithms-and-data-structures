## n個の頂点のそれぞれがほかのすべてのノードと辺で結ばれているグラフを頂点数nの完全グラフと呼び、Knと記す(Knがn個の頂点とn(n-1)/2 本の辺からなることに注意しよう)。グラフK6を図示し、その隣接行列と隣接グラフをそれぞれ示しなさい。ただし各辺のコストは1と仮定してよい。
- 隣接行列
  ```
  [0, 1, 1, 1, 1, 1],
  [1, 0, 1, 1, 1, 1],
  [1, 1, 0, 1, 1, 1],
  [1, 1, 1, 0, 1, 1],
  [1, 1, 1, 1, 0, 1],
  [1, 1, 1, 1, 1, 0]
  ```
- 隣接リスト
  ```
  [a] -> [b] -> [c] -> [d] -> [e] -> [f],
  [b] -> [a] -> [c] -> [d] -> [e] -> [f],
  [c] -> [a] -> [b] -> [d] -> [e] -> [f],
  [d] -> [a] -> [b] -> [c] -> [e] -> [f],
  [e] -> [a] -> [b] -> [c] -> [d] -> [f],
  [f] -> [a] -> [b] -> [c] -> [d] -> [e]
  ```
