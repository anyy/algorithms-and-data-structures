## ヒープに、6, 3, 4, 2, 5, 1, 7, 8 という要素をもつ8個のノードがこの順に挿入されたときの木の変化の様子を図示しなさい。ただし最初のノードが挿入される前のヒープはノードを含まないものとする。
-
  ```
  (1)
      [6]

  (2)
      [6]
       |
      [3]

  (3)
       __[6]__
      /       \
    [3]       [4]

  (4)
       __[6]__
      /       \
    [3]       [4]
     |
    [2]

  (5)
       __[6]__
      /       \
    [5]       [4]
    / \
  [2] [3]

  (6)
       __[6]__
      /       \
    [5]       [4]
    / \         \
  [2] [3]       [1]

  (6)
       __[7]__
      /       \
    [5]       [6]
    / \       / \
  [2] [3]   [1]  [4]

  (6)
       __[8]__
      /       \
    [7]       [6]
    / \       / \
  [5] [3]   [1]  [4]
   |
  [2]
  ```
