## 二分探索木に 6, 3, 4, 2, 5, 1, 7, 8 という鍵をもつ8個のノードがこの順に挿入されたときの木の変化の様子を図示しなさい。ただし最初のノードが挿入される前の二分探索木はノードをまったく含まないものとする。
```
           (1)[6|v|l|r]
                 /    \
      (2)[3|v|l|r]   (7)[7|v|l|r]
            /   \               \
  (4)[2|v|l|r]  (3)[4|v|l|r]   (8)[8|v|l|r]
         /        \
(6)[1|v|l|r]     (5)[5|v|l|r]
```
