## 2-3木に 6, 3, 4, 2, 5, 1, 7, 8 という鍵をもつ8個のノードがこの順に挿入されたときの木の変化の様子を図示しなさい。ただし最初のノードが挿入される前の2-3木はノードをまったく含まないものとし、最初の鍵6が挿入された時点でルートと葉ノードが一つずつ生成されるものとする。
```
(1)
      [ ]

(2)
      [ ]
       |
       6

(3)
      [ ]
      / \
     3   6

(4)
      _[ ]_
     /  |  \
    3   4   6

(5)
       __[ ]___
      /        \
     [ ]      [ ]
     / \      / \
    2   3    4   6

(6)
       ___[ ]___
      /         \
     [ ]       _[ ]_
     / \      /  |  \
    2   3    4   5   6

(7)
       ___[ ]___
      /         \
    _[ ]_      _[ ]_
   /  |  \    /  |  \
  1   2   3   4  5   6

(8)
       _________[ ]_________
      /          |          \
    _[ ]_       [ ]         [ ]
   /  |  \      / \         /  \
  1   2   3    4   5       6    7

(9)
       _________[ ]_________
      /          |          \
    _[ ]_       [ ]         _[ ]_
   /  |  \      / \        /  |  \
  1   2   3    4   5      6   7   8
```
